Pusat Tutorial & Keamanan

1. Ganti Nama

Buka Settings -> Edit Name. PayPal hanya mengizinkan ganti nama jika ada typo kecil atau pergantian nama legal (butuh upload KTP).

2. Ganti Alamat

Settings -> Addresses -> Add/Update. Alamat baru harus diverifikasi jika ingin dijadikan alamat utama penagihan.

3. Pengaturan Lain

Aktifkan 2FA di tab 'Security' untuk mencegah hack. Selalu ganti password setiap 3 bulan untuk keamanan ekstra.

4. Hindari Limit

Jangan pakai VPN. PayPal mendeteksi IP yang sering berubah. Hindari transaksi besar di akun baru tanpa verifikasi Bank.

Butuh Panduan Lebih Lanjut?
